ボタンの説明
C：1つ入力を戻す
AC:今までの計算結果を全て消去する

関数の説明
deci_cal(number):押下された数字入力用 number:int型
cal(operator): 演算処理&答え表示処理 operator:str型(+,-,×,÷)
